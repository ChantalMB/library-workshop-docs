### "and"
| Expression      | Evaluation |
| --------------- | ---------- |
| True and True   | True  |
| True and False  | False |
| False and True  | False |
| False and False | False |

### "or"
| Expression     | Evaluation  |
| -------------- | ----------- |
| True or True   | True  |
| True or False  | True  |
| False or True  | True  |
| False or False | False |
